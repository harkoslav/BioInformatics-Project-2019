# BioInformatics-Project-2019
Bioinformatics project for university project

## Compile & run:
`g++ -std=c++11 -Wall -I . ./*.cpp -o main.o `

`./main.o`


## Paralel execution
`g++ -std=c++11 -fopenmp -Wall -I . ./*.cpp -o main.o`

`./main.o`


## Score checking
To check the score run:
`make score`
